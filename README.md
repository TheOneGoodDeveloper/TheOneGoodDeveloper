### Full-Stack Engineer | Specialized in Scalable SaaS Architecture

I am a results-driven **MERN Stack Developer** with a focus on building production-grade, audited, and containerized applications. My approach combines engineering discipline with modern DevOps practices.

---

### 🛠️ Technical Arsenal

**Core Tech:** Node.js, Express.js, React.js, MongoDB (Advanced Aggregation)
**Enterprise Ready:** Redis (Caching), Docker (Containerization), JWT (Secure Auth + Refresh Tokens)
**Reliability:** Zod (Schema Validation), Jest (Unit Testing), MVC Architecture
**Observability:** Custom Audit Logging, Event Logging systems

---

### 🚀 Highlight Project: Dental Finance SaaS (MVP 2.0)

During my recent technical deep-dive (2024-2025), I architected a financial management platform for dental consultants to eliminate "revenue leakage" and automate complex payout structures.

* **Engineered for Speed:** Implemented **Redis** caching to handle sub-second financial reporting.
* **Built for Trust:** Developed an **Audit & Event Logging** system to track every financial transaction for 100% data integrity.
* **Cloud-Native:** Entire stack is **Dockerized** for seamless, environment-agnostic deployment.
* **Strict Security:** Integrated **JWT Refresh Token** logic and **Zod** validation to ensure zero-compromise data entry.
